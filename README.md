# PROOKT-FrontEnd-exam-202512
Prooktatás 2025 Szeptemberi k-cs Front-end beadandó
# Beadandó
7–10 napon belül egy egyszerű weboldalt kell készíteni, amely:
- esztétikus és áttekinthető
- teljesen reszponzív
- hamburger menüt tartalmaz
- legalább négy technikai elemet használ (pl. menü, táblázat, űrlap, kép/videó, JavaScript), nem alapulhat kész template-en
- Bootstrap használható, de nélküle jobb jegy érhető el
- lehetőleg több oldalból áll; one-page design esetén legfeljebb 4-es jegy adható
Határidő: január 5. (GIT-re feltenni)
